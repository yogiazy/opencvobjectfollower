Berikut adalah langkah instalasi file apk Robot Object Follower.

1. Download file object_follower.apk
2. Pastikan mengaktifkan terlebih dahulu fitur Unknown Sources atau
Sumber Tidak Diketahui pada pengaturan keamanan/security
3. Klik object_follower.apk untuk menginstall

Cara mengaktifkan fitur Unknown Sources atau
Sumber Tidak Diketahui pada pengaturan keamanan/security. 
Tiap jenis smartphone kadang berbeda, tetapi sebagian besar caranya seperti ini.

1. Buka Settings
2. Pilih Security
3. Pilih Unknown Sources
4. Jika muncul peringatan, pilih Ok
5. Selesai
